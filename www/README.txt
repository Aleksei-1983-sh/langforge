Полный проект (логин, регистрация, дашборд) с ассетами.

Как открыть локально:
- Откройте pages/login/index.html или pages/register/index.html в браузере.
- Для корректной работы API-запросов нужен сервер, который обрабатывает /api/* (login, register, me, logout).
- В текущей структуре пути к ассетам сделаны относительными (../../assets/...) — удобно для локального открытия.
